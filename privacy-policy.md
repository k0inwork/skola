Privacy Policy

Last updated: May 2026

This application ("Auto School") uses Google OAuth to provide authentication and access to Google services.

1. Data Accessed
The application may access:
- Google Calendar data (to create and manage lesson events)
- Google Mail (Gmail) sending capabilities (to send email notifications)

2. Purpose of Data Usage
Google Calendar access is used only to automatically create lesson events for the user.
Gmail access is used only to send email notifications related to lessons, scheduling, and reminders.

3. Data Storage
This application does not sell or share user data with third parties.
Google account data is used only to perform the intended application features.
If any data is stored, it is limited to what is necessary for scheduling and application functionality.

4. User Control
Users can revoke access at any time via their Google Account settings:
https://myaccount.google.com/permissions
